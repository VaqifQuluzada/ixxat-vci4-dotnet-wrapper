The online help for the VCI V4 .NET API is generated by using
Microsoft Sandcastle and the open source SandcastleBuilder.

The file vcinet2doc.shfb contains the SandcastleBuilder project. It
references the release verion of vcinet2.dll. So this has to be
built before generating the related help file.

Install the following software versions:
- SandcastelBuilder v2018.7.8.0
